============
Installation
============

At the command line::

    $ pip install 'git+ssh://git@bitbucket.org:dragons-astro/dragons.git'

Or, if you have virtualenvwrapper installed and want to test / maintain a
separate environment::

    $ mkvirtualenv dragons
    $ workon dragons
    $ pip install 'git+ssh://git@bitbucket.org:dragons-astro/dragons.git'
